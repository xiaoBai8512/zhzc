第一条    合作宗旨

本合同旨在约定乙方作为甲方平台服务的分销商，明确双方关于游客招徕和落实接待计划等事项的权利义务，保障双方及消费者的合法权益。

 

第二条    定义

1.       甲方平台：是指甲方拥有和运营的，通过移动客户端或网站方式为用户提供在线旅游产品服务的平台，包括但不限于[    ] 移动客户端及甲方发布的其他服务载体。

2.       相关规则：是指在甲方平台上发布和更新的各项服务规则文件，包括但不限于用户注册协议、交易规则等。

3.       平台服务：是指通过甲方平台向乙方提供的信息发布服务、订单管理服务等。

4.       商品：是指由甲方或供应商生产或提供的，经甲方审核通过并在甲方平台上进行展示、推广的各类旅游产品。

5.       分销商：是指经甲方平台审核通过，作为居间服务者，有权将展示在甲方平台上的商品向消费者进行推介并促成商品销售的平台用户。

6.       消费者：是指最终使用或消费甲方平台商品的终端用户。

7.       佣金：是指乙方每完成一次居间服务（即促成甲方平台的商品销售，使甲方平台获得消费者支付的费用，消费者有权使用商品），乙方有权根据甲方制定的标准按一定比例来获得其所提供的该次居间服务的报酬。

 

第三条    审核

乙方确认，其成为甲方平台的分销商需符合甲方平台规定的分销商入驻条件，并通过甲方平台申请页面提交申请，经甲方审核通过后才能成为甲方平台的分销商；甲方有权根据自己单方判断对乙方的申请进行审核评定。

 

第四条    服务内容

1.       乙方成为甲方平台的分销商后，享有以下甲方通过甲方平台提供的以下服务：

a)       可以在甲方平台开设乙方自己的店铺（店铺名称须经甲方审核通过方有效），并享受甲方提供的店铺运营的技术服务等。

b)       乙方可以通过其账户浏览所有通过甲方平台进行展示的供应商的商品信息，以及商品定价和商品佣金方案等，并可以将选定的商品上架到乙方商铺页面。

c)       乙方可以通过分享乙方店铺或店铺中商品的方式进行商品销售推介。

2.       双方确认，本条所述的甲方提供的服务内容具体以甲方平台系统实际能实现的功能为准，且甲方能够随时根据业务的发展需要或系统功能的调整等增加或修改相应的功能，或因定期、不定期的维护而暂缓提供，若因此导致系统迟延或为乙方带来的损失，甲方不承担任何责任。

 

第五条    合作流程

1.       乙方在甲方或供应商的授权范围内对展示在甲方平台的商品进行宣传推广，并通过甲方平台生成宣传推广链接，并通过其自有资源进行展示、推广和宣传。消费者通过乙方店铺形成的购买订单将由甲方平台收集和发送至供应商。

2.       乙方同意，经由其分销的消费者，甲方或其指定的第三方有权根据旅游产品的性质，决定由甲方、乙方或甲方指定的第三方与该消费者签订经甲、乙双方一致认可的旅游合同（“旅游合同”），乙方同意甲方或其指定的第三方届时的决定并无条件予以配合。根据《旅行社条例实施细则》的规定，游客有知情权，旅游合同中应明确提供旅游产品服务的旅行社名称、地址、电话、联系人等内容。乙方应在进行宣传推广过程中向该消费者合理披露前述情形，否则由此造成的一切责任均由乙方自行承担。

a)       若与消费者最终签署旅游合同的是甲方，则乙方应做好相应的信息披露及信息提交工作，积极配合核实并提供消费者的真实有效信息，否则由此造成的一切损失均由乙方自行承担，甲方有权自行对乙方的佣金予以抵扣。

b)       若与消费者最终签署旅游合同的是乙方，则乙方应按照《中华人民共和国旅游法》及相关法律法规的规定、及乙方与消费者签署的旅游合同约定提供相关服务，由此产生的一切纠纷与甲方无涉，均由乙方负责处理并承担责任；若因此给甲方造成损失的，甲方有权自行对乙方的佣金予以抵扣。

3.       消费者在行程中非因自身原因导致人身意外伤害或财产损失的，经有权部门确认责任归属的，由责任方承担相应的赔偿责任。因服务未达到约定的标准而导致游客或任何第三方投诉和/或索赔的，责任方按旅游合同约定的赔偿标准予以赔付；旅游合同未作约定的，按《旅行社质量保证金赔偿试行标准》的相应条款计算赔偿金额。

4.       消费者完成在线支付或根据相关合同约定以其他方式支付相应费用后，乙方可按照甲方平台的结算工规则获得本协议约定的佣金。

5.       消费者自行承担因其获得收而产生的税收，若根据法律法规需要甲方代扣代缴的则由甲方进行代扣代缴。

 

第六条    甲方的权利义务

1.       甲方根据本协议约定向法官提供信息及数据管理（包括但不限于信息发布、订单及采购单管理）服务。

2.       因网上平台的特殊性，甲方没有义务对所有分销商的信息、所有的交易行为以及有关的其他事项进行事项审查，但如存在下列情况：

a)       消费者或其他第三方通知甲方，认为乙方或具体交易事项可能存在重大问题;

b)       消费者或其他第三方向甲方告知乙方店铺上有违法或不当行为的；甲方通过自身判断认为该些内容或行为具有违反或不当性质的，甲方有权根据不同情况选择保留或删除相关信息或继续、停止对乙方继续提供分销商服务。

3.       甲方有权对乙方的基本信息及采购交易信息进行查阅，发现存在任何问题或怀疑，均有权向乙方发出询问及要求改正的通知或直接作出删除等处理。

4.       经国家生效法律文书或行政处罚决定确认乙方存在违法行为，或者甲方有足够事实依据可以认定乙方存在违法或违反本协议行为的，甲方有权根据具体情况决定是否继续提供本协议约定的服务。

5.       甲方或供应商展示在甲方平台的商品情况（包括但不限于旅游产品的线路、所含活动项目、服务标准、报价、出团计划书、注意事项等内容）应真实有效，基于行业特殊性的要求，且应至少包括酒店（住宿酒店名称、地点、档次等级等）、用餐、景点等信息。

6.       商品的售后服务由甲方负责，若乙方收到消费者用户关于商品售后问题的反馈意见，应立即告知甲方，由甲方进行沟通并转给供应商的售后服务部门解决。

 

第七条    乙方的权利义务

1.       乙方负责按照约定和相关法律法规宣传推介商品、招徕及组织参加旅游的消费者，因宣传商品产生的一切责任和费用均由乙方自行承担。乙方的工作人员应熟悉甲方的商品和业务操作程序，向消费者提供真实的商品资料并为消费者提供商品的咨询服务。

2.       乙方应根据甲、乙双方事先约定的标准和范围宣传商品的价格。未经甲方或供应商同意，乙方不得向消费者作出任何不实的承诺。

3.       若最终与消费者签署旅游合同的是甲方，则乙方应如实并谨慎地记录消费者的个人信息及参团要求，对消费者参加旅游的状态进行评估。因乙方原因（包括但不限于虚假陈述、放宽参团标准等情形）导致包括但不限于消费者在内的任何第三方投诉和/或索赔产生的损失均由乙方承担，若甲方先行向其他第三方进行赔偿或补偿的，乙方应当予补偿甲方，同时甲方有权就全部补偿和/或赔偿向乙方进行追偿。

4.       若最终与消费者签署旅游合同的是乙方，乙方须采用其主管部门（旅游管理部门）统一订制的格式文本，并提供与前述合同内容一致的发票。旅游合同中应明示甲方或供应商为组团社、说明旅游团的拼团性质，并提示应购买旅游者人身意外险等事宜。

5.       乙方应严格遵守本协议及甲方平台不时发布的相关规则，乙方同意该些规则一经发布生效即对其具有法律效力。

6.       乙方保证，如因乙方违反本协议约定，导致甲方受到行政执法机构的处罚（包括但不限于：停业整顿、处以罚款等）或被本协议外的任何第三方追究责任的，视为乙方严重违约，甲方有权选择解除本协议，乙方还应当协调与相关行政执法机构和本协议外第三方处理相关事宜，相关责任与甲方无关。在本款前述情形下，若甲方先行承担行政执法机构处罚或向本协议外第三方承担责任的，甲方有权在承担处罚和/或向本协议外第三方承担责任后全额向乙方追偿，乙方应当在收到甲方通知后的10日内，向甲方赔偿甲方被没收的费用、罚款、向其他第三方支付的费用、因处理争议所产生的律师费、误工费等全部费用和损失。

 

第八条    保证

1.       甲、乙双方保证遵守《中华人民共和国旅游法》、《中国公民出国旅游管理办法》等相关法律法规的规定，为消费者提供旅游服务。甲、乙双方保证无论在任何渠道发布商品的广告或进行宣传，其内容均应属实。

2.       甲、乙双方保证互相配合，本着严谨、周到的原则，按照双方确认的行程标准为提供旅游服务，履行旅游合同中的应尽义务。

 

第九条    佣金和结算

1.       若最终是由乙方与消费者签署旅游合同的，则乙方和甲方之间按照商品的结算价进行结算，具体结算价以甲方确定的结算规则为准。

2.       若最终是由甲方或供应商与消费者签署旅游合同的，具体佣金计算规则以甲方公布的为准。

3.       乙方在获得店铺时，必须绑定了一个开通了网络支付功能的有效银行账号或其他甲方认可的账户，户名必须与其用于实名制注册的手机号的名称一致。前述账号一经绑定即不得随意变更，乙方若需要变更账号，应当向甲方提交申请，并经甲方审核通过。在绑定的账号变更完成前，甲方有权暂停向乙方支付佣金。

4.       消费者用户取消交易，退货退款后，甲方在乙方账户扣除乙方已得到的相应的佣金。乙方佣金已经支取的，甲方可在乙方佣金账户内剩余佣金金额中扣除。如乙方佣金账户剩余佣金不足扣除累计180日的，视为乙方放弃本协议约定的分销商资格，甲方有权关闭并取消乙方分销商资格，同时保留向乙方追偿的权利。

g)        因不可抗力或甲方无法合理控制的其他原因所导致的损失。

4.       由于不能预见、不能避免、不能克服的客观原因，包括但不限于：地震、台风、海啸等自然灾害，政治动荡、暴动、战争、政府行为，以及通讯或其他设施故障等，导致一方无法履行本协议项下义务，不承担违约责任。

 

第十一条       保密条款

1.       甲、乙双方均承诺并保证不向第三方透露与本协议有关的的一切保密信息，该保密信息指由一方向对方透露的，与双方合作有关的任何口头或书面形式或其他形式的不公开信息，包括但不限于商业计划、客户名单、技术数据、产品构思、开发计划、职员名单、操作手册、加工工艺、技术理论、发明创造、财务情况和其他递交时约定为保密信息的资料（以下通称“保密信息”）。

4.       本保密条款不因双方协议终止而失效。

第十三条       法律适用及争议解决

1.       本协议的订立、变更、解除、解释、履行和争议的解决适用于中华人民共和国大陆地区法律。

2.       甲、乙双方在履行本协议过程中发生的任何争议与纠纷，应友好协商解决。协商不成的，甲、乙双方同意任何一方均应当将争议提交至北京仲裁委员会以仲裁方式解决，仲裁程序应依据该会届时有效的仲裁规则进行。仲裁地点在北京，仲裁裁决是终局的，对双方均具有约束力，双方均应遵照执行。

 

第十四条       服务终止

1.       本协议在下述情形发生时终止：

a)       双方协商一致，同意终止本协议的；

b)       除本协议另有约定外，乙方违反其他使用规范和相关规则，情节严重或造成重大损失，经甲方书面通知终止的；

c)       乙方不同意甲方对本协议或相关规则的修改，并且主动停止使用甲方平台的。

2.        发生下述情形时，甲方有权不经通知而中断、中止或终止向乙方提供服务，且无须承担任何责任：

a)       甲方认为乙方的登陆行为异常，存在账号被冒用风险的；
